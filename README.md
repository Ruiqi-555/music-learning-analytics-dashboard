# Music Learning Analytics Dashboard

A course project prototype for user behaviour analytics, user segmentation, and dashboard visualization.

This project simulates a music streaming membership system and provides a simple analytics dashboard for exploring user registration, listening behaviour, feedback, segmentation, and retention patterns.

## Project Context

This is a team course project.
This repository contains selected and cleaned project files for portfolio reference.

My main contributions focused on user behaviour analysis, segmentation logic, churn-risk interpretation, and analytical result presentation. The system prototype and front-end were developed collaboratively.

## Main Features

* Simulated user registration and listening behaviour data
* SQLite database schema for user, preference, listening log, feedback, and segmentation tables
* FastAPI backend for user registration, user search, data simulation, and dashboard generation
* Basic text mining for feedback keyword extraction and topic tagging
* User segmentation and churn-risk analysis
* Dashboard visualizations for user growth, DAU, retention, activity heatmap, and user segments
* Simple front-end pages for registration and admin dashboard interaction

## Tech Stack

* Python
* FastAPI
* SQLite
* Pandas
* Matplotlib / Seaborn
* HTML / Tailwind CSS
* JavaScript

## Notes

Raw generated database files, output charts, and large intermediate files are not included in this repository.

The project uses simulated data generated with Python scripts. No real user data is included.
